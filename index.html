What is VeriSPro?

VeriSPro stands for “Verifying Security Protocols”.

VeriSPro includes researchers from INRIA Paris, IIT Gandhinagar and IIT Delhi. The objective of this work is to use the DY* verification framework to prove security properties of protocols.

In the first phase, we focus on group messaging protocols used in applications like Signal, WhatsApp. We want to prove the security of these protocols in the DY* framework, which is a formal verification framework for the symbolic security analysis of cryptographic protocol code written in the F* programming language.
